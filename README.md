# 🚀 CRUD de Projetos com MongoDB

Sistema simples de cadastro, listagem e exclusão de projetos, utilizando **MongoDB** como banco de dados.

## 📌 Funcionalidades

* ✅ Cadastrar novo projeto
* ✅ Listar projetos salvos
* ✅ Excluir projetos
* ✅ Integração com MongoDB Atlas
* ✅ Interface simples e direta

---

## 🛠️ Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla)**
* **Node.js**
* **MongoDB Atlas**

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/seu-repo.git
```

---

### 2. Instale as dependências

```bash
npm install
```

---

### 3. Configure o MongoDB

Crie um arquivo `.env`:

```env
MONGO_URI=sua_string_de_conexao
```

---

### 4. Rode o projeto

```bash
npm start
```
